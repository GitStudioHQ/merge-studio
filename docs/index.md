# Merge Studio

**JetBrains-style merge & diff for VS Code and Cursor.** Resolve merge conflicts
the way IntelliJ, PyCharm and WebStorm do it: a three-pane merge editor with
gutter ribbons, an undo history, and a Conflicts dialog — plus a side-by-side
diff viewer.

## Install

- **Open VSX** (Cursor, VSCodium): [open-vsx.org/extension/gitstudio/merge-studio](https://open-vsx.org/extension/gitstudio/merge-studio)
- **VSIX**: grab the latest from [GitHub Releases](https://github.com/antonarnaudov/merge-studio/releases) and run *Extensions: Install from VSIX…*

## What it does

- **3-way merge editor** — Left (yours), Result, Right (theirs), with accept /
  ignore ribbons per change and a per-action history.
- **Conflicts dialog** — every conflicted file in one list, with Accept Yours /
  Accept Theirs / Merge and a hold-to-undo.
- **Side-by-side diff** — working tree vs HEAD, JetBrains-style.
- **Open in your JetBrains IDE** — hand any conflict to the real
  IntelliJ / PyCharm / WebStorm merge window when you'd rather finish it there.
